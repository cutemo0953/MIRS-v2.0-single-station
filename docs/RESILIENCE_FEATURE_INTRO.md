# MIRS 韌性估算功能介紹

> 🏥 偏遠醫療站的生命線管理系統

---

## 一句話說明

**在孤立無援時，你的醫療站還能撐多久？**

韌性估算功能讓你一眼掌握：氧氣、電力、物資能支撐多少小時，哪個環節最脆弱。

---

## 核心功能

### 🔋 生命線追蹤
- **氧氣供應**: 鋼瓶存量 × 充填% ÷ 病患需求
- **電力供應**: 發電機油量 + 電源站容量 ÷ 設備負載
- **可獨立運作時數**: 取兩者最短值

### 📊 每日設備檢查
- 樹狀展開介面，每支氧氣瓶獨立追蹤
- 充填百分比一目瞭然
- 未檢查項目灰顯提醒
- 檢查進度 (3/5 已檢查)

### 📋 檢查報表
- 每日檢查記錄自動保存
- 可選日期查看歷史
- CSV 匯出供稽核

### ⚡ 即時計算
- 修改設備狀態，韌性時數即時更新
- 調整病患人數，氧氣需求自動換算
- 設定孤立天數目標，顯示達標狀態

---

## 使用情境

```
情境：颱風警報，可能斷路 3 天

1. 打開韌性估算 Tab
2. 設定「預估孤立天數: 3」
3. 查看當前可獨立運作時數
4. 發現只有 48 小時 → 不足！
5. 檢查哪個生命線最短
6. 補充物資或調整病患安排
```
